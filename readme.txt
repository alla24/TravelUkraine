Travel Ukraine
~~~~~~~~~~~~~~~~~~~~

The application provides information about different locations in Ukraine. The target audience comprise english speaking travelers.

Install:

For installing the app you need to modify your Android's settings to allow the installation of applications from other sources. Under Settings select Security and then enable Unknown Sources.
Copy the APK file to your Android's memory. Click and install the APK file.


After the application is launched the user shall see main window (app logotype, name and list of locations) allowing to scroll and choose location of interest.
After the choice is made and city is selected the application shows all basic information about the city of choice (name, area, population, description).
Additionally it allows view positioning of the city on map after clicking the button 'Show on map' which redirects user to map application (e.g. Google Maps).
